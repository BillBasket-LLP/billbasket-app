# BillBasket

**Offline-first billing, POS & inventory app for small shops.**
Works on Windows desktop and Android — no internet required to run your business.

---

## What is BillBasket?

BillBasket is a complete billing solution built for small and medium shops in India and South Asia. It runs fully offline on your Windows PC or Android phone, keeps your data encrypted, and optionally syncs to the cloud when you need it.

- **Fast billing** — create GST/non-GST invoices in seconds with a barcode scanner or typeahead search
- **Inventory tracking** — batch/serial numbers, stock adjustments, low-stock alerts, COGS reports
- **Parties (customers & vendors)** — ledgers, balances, payment history, aging report
- **Purchases & expenses** — vendor invoices, AI-assisted receipt parsing, expense categories
- **Estimates** — convert to invoice in one tap
- **Reports** — Z-report, day book, receivables/payables, sales velocity, COGS, inventory valuation
- **Web access** — share live data with a phone or browser on the same Wi-Fi (no cloud needed)
- **AI assistant** — on-device AI for invoice drafting, vendor invoice parsing, payment reminders (optional)
- **16 languages** — English, Hindi, Bengali, Marathi, Telugu, Tamil, Gujarati, Kannada, Odia, Malayalam, Punjabi, Assamese, Nepali, Mizo, Spanish, French

## Installation (Windows)

1. Download `BillBasket-Setup-1.0.83.exe` from the table above.
2. Run the installer — it does not require administrator rights.
3. On first launch, create a PIN and set up your store. No account needed to start.
4. Optional: sign up at [accounts.billbasket.app](https://accounts.billbasket.app) to enable cloud backup and cross-device access.
